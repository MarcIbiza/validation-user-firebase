# firebase-auth

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
### Instalar firebase
```
npm install -g firebase-tools
```

### Login firebase
```
firebase login
```

### Iniciar firebase
```
firebase init
```

### Auditar ficheros antes del buil y resolver conflictos
```
npm audit
```
npm audit fix

### Compiles and minifies for production
```
npm run build
```

### Hacer el deploy
```
firebase deploy
```
